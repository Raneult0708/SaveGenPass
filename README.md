#  Générateur de mots de passe

##  Description
Ce projet est un **générateur de mots de passe sécurisé** développé en Python.  
Il permet à l’utilisateur de créer des mots de passe personnalisés en choisissant les caractéristiques souhaitées (longueur, types de caractères, etc.).

---

##  Objectifs
- Générer des mots de passe robustes
- Permettre une personnalisation complète
- Sensibiliser à la sécurité informatique
- Proposer un outil simple et efficace

---

##  Fonctionnalités
- Choix de la longueur du mot de passe
- Inclusion de :
  - Lettres majuscules
  - Lettres minuscules
  - Chiffres
  - Caractères spéciaux
- Génération aléatoire sécurisée
- Affichage immédiat du mot de passe

---

##  Technologies utilisées
- Python 3
- Bibliothèques standard :
  - `random`
  - `string`
